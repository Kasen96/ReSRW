# ReSoW

ReSoW is a simple application that read data from the disk, process the data (sorting) and then write back the data on the disk.

# Baseline

* Sorting algorithm = insertion sort
* Standard read/write buffer size = 102400
* Read/write block size equal to the size of a single record in the file, that is a floating point number.
* gcc compiler optimization turned off
